### session 09 (wed 6/17)

- started JavaScript for real.
- the script tag goes at the bottom of body so the page content loads before the js runs.
- `addEventListener` is the recommended way to bind events. inline `onclick="..."` works but mixes concerns, same problem as inline css.
- `Math.random()` returns a float between 0 and 1. multiply by n to get a range, then `Math.floor()` to get an integer. that's how you grab a random index from an array.
- `querySelector` uses css selectors and is more flexible. `getElementById` only works on ids and is slightly faster but less useful.
- events are signals the browser fires. click, mouseover, mouseout, keydown, keyup, input, submit, dblclick. mobile has its own: touch, pointer, devicemotion.
- dom is document object model. tree structure. parent and descendants.
