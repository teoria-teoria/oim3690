## what i learned in class

session 04 (monday 6/1) was css. main things:
- three ways to add css. external file is the right way. internal `<style>` block is okay for one-pagers. inline is a hack.
- the cascade. when two rules conflict, more specific wins. element < class < id < inline. ties broken by whatever comes later.
- selectors: element (`h1`), class (`.card`), id (`#header`), pseudo-class (`:hover`), descendant (`nav a`).
- inheritance. set font-family on body once, the whole page picks it up.
- devtools is for inspecting elements and reading their applied css, not just for the network tab.
